# Modern Java (17/21/25) Mini-Topics

This folder adds short, practical topics to help students get familiar with modern Java features.

## Why LTS?

- Java 17 and Java 21 are Long-Term Support (LTS) releases.
- Java 25 is a non-LTS release (shorter support window).
- In industry, LTS is often chosen for stability and long-term updates.

## Topics Covered

- `var` local variable type inference
- `instanceof` pattern matching
- String updates: `strip`, `isBlank`, `lines`, `repeat`
- `Optional` basics

## How to run examples

From the `src` folder:

```
# Compile
javac modernjava/examples/*.java

# Run one example
java modernjava.examples.ExampleVar
```

- `ExampleVar.java`
